## 🔬 Recent / Ongoing Work  

✈️ **Predictive Anomaly Detection on Aviation Sensor Data (NGAFID)**  
Built a hybrid ML + deep learning system for early aircraft fault detection using One-Class SVM and neuroevolutionary RNNs (EXAMM) on real-world flight sensor data. Focused on time-series forecasting, anomaly scoring, and model interpretability for predictive maintenance.

🧘 **Real-Time Yoga Pose Detection App (Computer Vision + Deep Learning)**  
Developed a Flask-based web app using TensorFlow and MediaPipe to classify yoga poses in real time from webcam input. Provides live form feedback and was prototyped during BrickHack 2025.

❤️ **Heart Disease Diagnosis Using Hybrid Fuzzy Membership Models**  
Designed an interpretable fuzzy-rule-based classifier in Python to improve diagnostic accuracy on medical datasets. Combined fuzzy membership functions with decision logic to balance performance with clinical transparency.

🚦 **U.S. Traffic Fatality Analysis with BigQuery & Looker Studio**  
Analyzed NHTSA crash data using advanced SQL and BigQuery ML to uncover behavioral and environmental risk factors. Built interactive Looker Studio dashboards and logistic regression models to predict multi-fatality crash risk and visualize geospatial patterns.
